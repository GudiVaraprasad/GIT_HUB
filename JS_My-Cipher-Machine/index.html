How this works ?

This Simple Caesar Cipher Engine is probably one of the most basic ciphers, although it was the basis of the Cipher Encryption. The entered text will be encrypted based on shift factor (key value). Method in which each letter in the plaintext is replaced by a letter some fixed number of positions down the alphabet. The method is named after Julius Caesar, who used it in his private correspondence.

The cipher I have used is the Caesar cipher, named after the Roman emperor, Julius Caesar. It is one of the most simple ciphers there are and simply shifts each letter along a set number of places. For example, the phrase ‘Hello World’ would become ‘KHOOR ZRUOG’ using a shift of key value=3 (which it is the shift that Julius Caesar is thought to have used).
